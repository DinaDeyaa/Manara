# Manara

Manara is an AI-based personalized course path recommendation system developed for PSUT undergraduate students.

## Project Overview
The goal of Manara is to support students in planning their academic paths by analyzing their academic history, including completed courses, grades, and attendance. Based on this analysis, the system provides personalized recommendations that align with degree requirements and student readiness.

## Repository Contents
- dataset_generation.ipynb  
  Jupyter Notebook used to generate the synthetic dataset and perform preprocessing.

- students.csv  
  Contains student profile information and cumulative GPA.

- grades.csv  
  Contains course-level grades for each student.

- attendance.csv  
  Contains attendance records per course.

## Dataset Notes
Real student data was not available due to privacy constraints. Therefore, a synthetic dataset was generated using Python and the Faker library to simulate realistic academic performance patterns consistent with PSUT regulations.
